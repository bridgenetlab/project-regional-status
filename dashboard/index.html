🔑 Setup Google Maps API
To use the map features, you'll need a Google Maps API Key. Here's how to get one:
1 Go to Google Cloud Console
2 Create a new project (or select existing)
3 Enable Maps JavaScript API and Geocoding API
4 Create an API key in the Credentials section (click "Create Credentials" → API Key)
